This is a Javascript For Loop

A for loop in any programming language, goes through a block of code until the loop is told to end. In the code below, I have written a for loop to count from 1 to 10. For every number the loop counts, i've used the javascript framework "Jquery", to find the div with the id of "Result" and append a span with the css class of "Scale". The the scale class will utilize css3 keyframe animations to scale each number to blow them up a bit then reduce them back down to their original size by the end of the animation. You'll also notice there is a self invoked function wrapped around each time the class of "scale" is appended to the div with the id of "result". This function sets a time out so each number will appear one after the other and not all at once.
